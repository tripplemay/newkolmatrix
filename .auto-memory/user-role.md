用户是 KOLMatrix 的**产品负责人**（技术型，能读懂框架差异、Prisma、Next.js 等技术细节）。

- 使用 Claude Code 配合 Triad Workflow 7 状态机（new → planning → building → verifying → fixing ⟷ reverifying → done）进行迭代开发
- 主上下文负责 planning / building / fixing / done，隔离 evaluator subagent（或独立实例）负责 verifying / reverifying
- 技术栈熟悉：Next.js / React / TypeScript / Tailwind
- 工作语言：中文沟通，代码 / commit 用英文
- 偏好：**给方案时要带背景解释与方案权衡（trade-off），不要只丢结论**；从产品定位和用户体验视角审视决策
- 关注点：产品定位是"AI 驱动的 KOL 营销平台"，交互与流程需体现 AI-native，不走传统 SaaS 老路
